# Paths 模块

路径系统模块，提供复杂路径的创建、编辑和渲染功能。

## 主要组件

### LinearPath
线性路径，连接多个点的直线路径。

**功能：**
- 折线路径创建
- 路径长度计算
- 点在路径上的插值
- 路径简化算法

### BezierPath
贝塞尔曲线路径，支持平滑曲线。

**曲线类型：**
- 二次贝塞尔曲线
- 三次贝塞尔曲线
- 复合贝塞尔路径
- 自适应细分

### PathFactory
路径工厂，快速创建常用路径形状。

**预设路径：**
- 圆形路径
- 椭圆路径
- 星形路径
- 螺旋路径

## 路径操作

### 路径编辑
- **点添加/删除**：动态修改路径
- **曲线控制**：贝塞尔控制点编辑
- **路径合并**：多条路径组合
- **路径分割**：路径切分操作

### 路径计算
- **长度测量**：精确路径长度
- **切线计算**：路径切线方向
- **法线计算**：路径法线方向
- **曲率分析**：路径弯曲程度

## 使用示例

```typescript
import { LinearPath, BezierPath, PathFactory } from './paths';

// 创建线性路径
const linearPath = new LinearPath([
  { x: 0, y: 0 },
  { x: 100, y: 50 },
  { x: 200, y: 100 }
]);

// 创建贝塞尔路径
const bezierPath = new BezierPath([
  { x: 0, y: 0 },
  { x: 50, y: -50 },  // 控制点
  { x: 150, y: 50 },  // 控制点
  { x: 200, y: 0 }
]);

// 使用工厂创建圆形路径
const circlePath = PathFactory.createCircle(
  { x: 100, y: 100 }, // 中心点
  50                  // 半径
);

// 获取路径上的点
const point = linearPath.getPointAt(0.5); // 中点
const tangent = bezierPath.getTangentAt(0.3);
```

## 路径动画

### 路径跟随
- **对象路径动画**：对象沿路径运动
- **速度控制**：匀速或变速运动
- **方向对齐**：对象朝向路径切线
- **循环路径**：无限循环动画

### 路径变形
- **路径插值**：两条路径间过渡
- **路径弯曲**：动态改变路径形状
- **路径扭曲**：特殊变形效果
- **噪声扰动**：添加随机扰动
