# Editor 模块

场景编辑器模块，提供可视化的场景编辑和调试功能。

## 主要组件

### SceneEditor
场景编辑器，提供交互式的场景编辑功能。

**主要功能：**
- 对象选择和变换
- 实时属性编辑
- 场景层次管理
- 撤销/重做功能

## 编辑功能

### 对象操作
- **选择工具**：点击选择场景对象
- **变换工具**：移动、旋转、缩放操作
- **复制粘贴**：对象复制和粘贴
- **删除操作**：安全删除场景对象

### 属性编辑
- **实时预览**：属性修改的实时反馈
- **批量编辑**：多对象同时编辑
- **属性面板**：可视化属性编辑界面
- **数值输入**：精确的数值设置

## 使用示例

```typescript
import { SceneEditor } from './editor';

// 创建场景编辑器
const editor = new SceneEditor(canvas, scene);

// 启用编辑模式
editor.enableEditMode(true);

// 选择对象
editor.selectObject(targetObject);

// 编辑对象属性
editor.setProperty('position', { x: 100, y: 200 });
```

## 交互功能

### 鼠标操作
- **左键选择**：选择场景对象
- **拖拽移动**：直接拖拽对象位置
- **右键菜单**：上下文操作菜单
- **滚轮缩放**：场景视图缩放

### 键盘快捷键
- **Ctrl+Z**：撤销操作
- **Ctrl+Y**：重做操作
- **Delete**：删除选中对象
- **Ctrl+C/V**：复制粘贴
